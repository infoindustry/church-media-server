
## v1.2

- Убран большой JSON/debug-блок с главной страницы админки.
- Вместо JSON теперь показывается короткая человеческая сводка текущего экрана.

# Changelog

## v0.9.0

- Добавлен выбор места Писания через селекты: книга → глава → стих/диапазон стихов.
- Оставлен ручной ввод ссылки: `Ин 3:16`, `Иоанна 7:37-38`, `John 3:16`.
- Исправлен вывод текста Писания на TV-экран: теперь payload содержит `blocks` и fallback `text`.
- TV-экран Писания теперь показывает 1, 2 или 3 перевода крупно и с переносами строк.
- Добавлен API `/api/bible/books` для списка книг, глав и количества стихов.
- Проверена production-сборка: `npm run build` проходит успешно.

## v0.8.0

- Добавлена команда `npm run import:serbian`.
